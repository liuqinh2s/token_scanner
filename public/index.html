🔍 扫描流程

1
链上发现 (~1s)
BSC RPC eth_getLogs 扫描 four.meme TokenCreated 事件，100% 覆盖链上新代币
2
入场筛选 (~数秒)
four.meme Detail API → 淘汰无社交 / 总量≠10亿 / 币龄过大
3
队列淘汰 (~数秒)
DexScreener 批量查价 + GeckoTerminal 持币数 + Detail API → 永久淘汰弃盘币
4
动量精筛 (瞬时)
从存活币中找起飞信号：价格动量 + 持币增长 + 买卖比 + 进度 + 回撤保护
5
精筛后防线 (~数秒)
Top10 持仓集中度 + 开发者行为分析 + 假K线检测 → 排除庄家控盘、跑路币和全阳线控盘币
6
仿盘检测 (零 API)
本地统计同名代币数量，≥3 个同名标记为仿盘（仅标记，不排除）

❌ 淘汰条件

  • 价格从峰值跌 90%+
  • 持币数从 ≥30 跌破 10
  • 持币数从峰值跌 70%+（峰值≥50，清理僵尸币）
  • 无社交媒体
  • 流动性从 >$1k 跌破 $100（仅已毕业代币）
  • 进度 < 1% 且币龄 > 2h
  • 进度 < 5% 且币龄 > 4h
  • 进度从峰值跌 50%+ 且币龄 > 6h（热度消退）
  • 币龄 > 15min 且最高持币 < 3
  • 币龄 > 1h 且最高持币 < 5
  • 币龄 > 48h

✅ 精筛条件（全部满足）

  • 币龄 ≥ 15 分钟（数据稳定后再判断）
  • 持币地址数 ≥ 15
  • 价格动量: 当前价 ≥ 入队价 × 1.5 或 ≥ 历史最低价 × 2.5
  • 持币增长: 当前持币 ≥ 入队持币 × 1.5 或近 3 轮持续递增
  • 进度 ≥ 20% 且 < 97%（低进度表现差；接近毕业/已毕业买入即亏）
  • 进度从 10%+ 跌破 5% 排除（衰退信号）
  • 已毕业代币流动性 ≥ $500
  • 买卖比 ≥ 1.2（买压 > 卖压，有人在吸筹）
  • 价格加速度 ≥ 5%（最近 2 轮价格上涨速率）
  • 持币增速 ≥ 10%（最近 2 轮持币增长速率）
  • 回撤保护: 当前价 ≥ 峰值 × 0.5
  • 精筛冷却: 通过后 6 轮内不再重复推送

🛡️ 精筛后防线（深度检查）

仅对精筛通过的少量代币执行，不影响扫描速度

  • Top10 持仓占比 ≤ 85%（排除庄家控盘）
  • 开发者行为: 清仓/撤池子 → 排除（跑路信号）
  • 假K线检测: 排除全是阳线实体柱、无影线、无阴线的控盘币（15min K线无影线 ≥ 80% 或全阳线 ≥ 90%）

📊 数据源

  • BSC RPC — 链上 TokenCreated 事件发现
  • four.meme Detail API — 代币详情 / 社交 / 进度 / 未毕业持币数
  • DexScreener — 批量价格 / 流动性 / 交易量 / 买卖笔数
  • GeckoTerminal — 已毕业代币持币数 + 15min K线（假K线检测）
  • BSCScan (Etherscan V2) — Top Holder 集中度 + 开发者行为分析（仅精筛后防线）
